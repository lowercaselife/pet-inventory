# Pet Inventory Demo (TypeScript + Next.js)

This repository is a demo to show TypeScript in React (Next.js).

## Setup
- Ensure Node.js is installed
- Install dependencies:

```bash
npm install
```

## Development
- Start the dev server:

```bash
npm run dev
```

- Open `http://localhost:3000` in your browser

## Production
- Build the app:

```bash
npm run build
```

- Start the production server:

```bash
npm run start
```

## Linting
- Run ESLint:

```bash
npm run lint
```
